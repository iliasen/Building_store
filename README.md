# Building_store

## Tech Stack
* React
* MobX
* JavaScript
* Styled-components
* Node.js
* PosgreSQL

## How to run the app

1. Clone this repository by running `$ git@gitflic.ru:iliasen/building_store.git`
2. Run `cd server`
3. Run `npm i`
4. Run `npm run dev`
5. Open secound window of termenal
6. Run `cd client`
7. Run `npm i`
8. Run `npm start`
